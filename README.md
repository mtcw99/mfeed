# mfeed
Mart's RSS Feed Reader

## Requirements
* C++17/20
* [libcurl](https://curl.haxx.se/libcurl/)
* [cmake](https://cmake.org/)

### Dependencies (cmake auto downloads)
* [fmt](https://github.com/fmtlib/fmt)
* [pugixml](https://github.com/zeux/pugixml)

## Build
* `mkdir build`
* `cd build`
* `cmake ..`
* `cd src`
* `make`
